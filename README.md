# Cyclone Impact Analysis & Clustering

A beginner-friendly Machine Learning project that analyzes cyclone-related data and groups cyclones based on their characteristics using K-Means Clustering.

This project was created to explore how different environmental factors like pressure and wind speed affect cyclone behavior and impact.

---

## About The Project

Cyclones are one of the most dangerous natural disasters.
Their impact depends on several factors such as:

* Wind Speed
* Atmospheric Pressure
* Rainfall
* Storm Surge
* Ocean Temperature

In this project, I used Machine Learning techniques to:

* Process cyclone data
* Normalize features
* Cluster similar cyclones
* Analyze cyclone categories

The main goal of this project is learning and experimenting with real-world ML workflows using Python.

---

## Features

* Data preprocessing using StandardScaler
* Cyclone grouping using K-Means Clustering
* Dataset analysis with Pandas
* Simple ML workflow for beginners
* Custom cyclone dataset

## Machine Learning Used

Main preprocessing steps:
